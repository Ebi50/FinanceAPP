# **App Name**: ExpenceTrack

## Core Features:

- Secure Authentication: Secure login system with role-based access control (Admin/User).
- Real-Time Expense Dashboard: Live expense tracking with visual budget progress, offering real-time insights into spending habits.
- Smart Transaction Manager: Enables users to easily add, edit, and categorize transactions, utilizing smart suggestions to streamline the process and ensure accuracy. Multiple amounts can be added to the same category during input.
- Dynamic Category Management: Complete CRUD operations for expense categories and subcategories. All buttons and labels are in German.
- Automated Category Suggestions: An AI-powered tool suggests appropriate categories for new transactions based on descriptions.
- Currency Handling: Facilitates proper German locale formatting for Euro currency (€).
- Offline Support: Enable the application to function while the device is disconnected from the internet.
- Excel Import: Allows uploading expense data from Excel files.
- PDF Reports: Generates monthly and yearly expense reports in PDF format.
- Transaction Modification: Allows modification of all transaction details.
- Recurring Transactions: Transactions can be marked as recurring monthly, with the ability to change the amount for future or past occurrences on a specific day.

## Style Guidelines:

- Primary color: Desaturated cyan (#A7D1AB) for a calm, trustworthy feel.
- Background color: Dark scheme with a hue similar to the primary color, offering a clean and modern interface (#1A202C).
- Accent color: Yellow (#FFDA63), an analogous color that is brighter than the primary and contrasts with the background for highlights and key actions.
- Headline font: 'Space Grotesk' (sans-serif) for headlines, giving a modern, computerized feel.
- Body font: 'Inter' (sans-serif) for body text, offering a neutral, readable style. Note: currently only Google Fonts are supported.
- Code font: 'Source Code Pro' (monospace) for code snippets. Note: currently only Google Fonts are supported.
- Use modern icons from 'Lucide React' to maintain consistency.
- Implement subtle transitions and animations using framer-motion.
